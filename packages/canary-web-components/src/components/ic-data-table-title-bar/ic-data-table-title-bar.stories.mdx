# [Canary]: Data Table Title Bar

Use the Data Table Title Bar component to provide titling, description, metadata and also global table functionality.

```html
<ic-data-table id="data-table" caption="Custom Title Bar">
  <ic-data-table-title-bar
    description="Data table description that describes the purpose of the table."
  ></ic-data-table-title-bar>
</ic-data-table>
```

### Basic

A basic example with a heading.

By default, the density selector appears within the Data Table Title Bar. The three density options are:

- Default
- Dense
- Spacious

It is also possible to pass custom elements into the area which contains the title bar heading and description. This flexibility allows the heading tag to be updated if the heading prop is not semantically corrected based on your content structure.

### Hide density selector

By default, the density selector is displayed on the Data Table Title Bar. To hide it, set `hide-density-select` to `true`.

<Canvas withSource="none">
  <Story name="Hide density selector">

      hide-density-select="true"
      heading="Title Bar"
      metadata="128 items | 32gb | Updated: 01/02/03"
      description="Data table description that describes the purpose of the table."
    ></ic-data-table-title-bar>`}
  </Story>
</Canvas>

```html
<ic-data-table-title-bar

```

### Custom actions

Custom actions can be added to the Data Table Title Bar via slots.

There are two slots available:

- `primary-action` is suitable for an primary button. The custom element will appear to the right of the Data Table Title Bar.
- `custom-actions` can be used for secondary actions. The custom element will appear to the left of the density selector if displayed.
