### Component overrides

If a slot is provided, instead an attribute, the slot will supersede for the following:

- heading
- subheading

<Canvas>
  <Story name="Component overrides">
    {html`
      <ic-page-header>
        <ic-typography slot="heading" variant="h4">
          <h2>heading</h2>
        </ic-typography>
        <ic-typography slot="subheading" variant="body">
          subheading
        </ic-typography>
      </ic-page-header>
    `}
  </Story>
</Canvas>
